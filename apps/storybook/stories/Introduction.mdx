# UI Component Library

Welcome to the **@workspace/ui** component library documentation! This Storybook instance showcases all the reusable UI components built with:

- **React 19** - Modern React with latest features
- **TypeScript** - Type-safe component development
- **Tailwind CSS v4** - Utility-first styling
- **shadcn/ui** - Beautifully designed components
- **Radix UI** - Accessible component primitives

## Getting Started

All components are available from the `@workspace/ui` package and can be imported directly:

```tsx
import { Button } from "@workspace/ui/components/button";
import "@workspace/ui/globals.css";

function MyApp() {
  return <Button>Click me</Button>;
}
```
